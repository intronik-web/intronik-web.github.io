Getting Started

To set up your website on Intronik, follow these guidelines:

  • For a single-page website, simply upload your config.json file to the root directory.
  • If your site has multiple pages, organize them into separate folders. Ensure that each folder contains a config.json file to maintain consistency with the rendering.
  • All config.json files must be hosted on a separate server URL. Make sure the URLs are correctly referenced in your website setup.
  • To render your website, use the following URL format: https://intronik-web.github.io/render?config=https://url.to/your/config.json. Replace https://url.to/your/config.json with the actual URL of your configuration file.

This structure ensures that your website's configuration remains organized and functional across different pages and is easily accessible from your hosting setup.
