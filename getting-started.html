Getting Started with Intronik

Ready to set up your website with Intronik? Follow these simple steps:

Step 1: Create Your config.json

Start by creating your config.json file. This file contains all the configuration details for your website.

Need help with creating your config.json? Visit our Config Maker tool to easily generate one.

Step 2: Upload Your config.json

Upload your config.json file to the root directory of your hosting setup. If you have multiple pages, make sure each directory has its own config.json.

Step 3: Access Your Website

To render your website, use the following URL format:

https://intronik-web.github.io/render?config=https://url.to/your/config.json

Replace https://url.to/your/config.json with the actual URL of your configuration file.

Step 4: Enjoy!

That's it! Your website should now be live and ready to go. If you encounter any issues, feel free to reach out to our support team.
